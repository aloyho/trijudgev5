How to Use TriJudge
Quick Guide
ENTRY SCREEN
Choose Battle Judge for head-to-head battles, or Open/Kpop Cover Showcase for group performance scoring.
BATTLE — SCORE MODE
Select dancer count (1, 2, 2v2, 3v3, 5v5). For team battles (2v2, 3v3, 5v5), one scorecard represents each team — D1 and D2. Set score scale (default 10). Tap dots to score on Body, Mind, Soul. Press Lock Score to save. Each entry gets an auto ID (#1, #2...). Tap any saved entry to edit.
BATTLE — WEIGHTAGE MODE
Enter dancer names. Drag the crossfaders left/right for each criterion to indicate which dancer won that category. Lock In Verdict to save the round to history.
SHOWCASE MODE
Choose Open or Kpop type. Tap Edit Criteria to toggle or add custom criteria. Set Weightage to assign % to each criterion (must total 100%). Score each group using the Slider (drag to set) or Direct Key-in mode. Each criterion's max score equals its weight % — e.g. Creativity at 20% means you score 0–20. The total out of 100 is calculated automatically. Then Lock Score.
EXPORT / SHARE
Tap the + button → Share / Export to view and copy all results. You can also export as CSV for spreadsheet use.
OFFLINE USE
Install TriJudge to your Home Screen for full offline access. All data is saved locally to your device.
